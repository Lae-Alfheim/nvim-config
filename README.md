# Install

git clone this repo, then install
(packer)[https://github.com/wbthomason/packer.nvim#quickstart] if not
already installed.


Then Run 

    :PackerInstall

To Download all the plugins


Then you need to install (nerd font)[https://www.nerdfonts.com/] to have the
full Experience
